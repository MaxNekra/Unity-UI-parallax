********************************************
************* ASSET INFORMATION ************
********************************************

Version: 1.0
Supported Unity Versions : 5.5.0 and above.

Description:
************
	This asset provides a parallax system based on the UI.


********************************************
********* DESCRIPTION OF SYSTEMS ***********
********************************************

1. FollowParallax - The system in which the parallax moves in the direction of the object (target).
Example: When a ship flies in space.

2. ConstantParallax - This system moves the parallax in the specified direction.
Example: Back menu background


********************************************
************** GETTING STARTED *************
********************************************

All examples you can find in the folder Example.

1. Create a Canvas.
2. Create and configure RawImage.
3. Choose a suitable parallax system for you. (FollowParallax or ConstantParallax).
4. Place a script on any object and configure it.
